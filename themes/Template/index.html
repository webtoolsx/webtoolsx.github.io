This project is a responsive Profile Page built using HTML, CSS, and Bootstrap 5. It includes sections for user details, skills, portfolio, and contact information, making it an excellent template for personal or professional use.

Features

  • User Profile Section – Profile picture, name, bio, and job title.
  • Skills & Experience – Progress bars to showcase skills.
  • Portfolio/Gallery – A responsive grid or image slider.
  • Contact Information – Email, phone, and social media links.
  • Dark/Light Mode Toggle – Allows users to switch themes.
  • Modern Design – Uses Bootstrap’s latest components.

Technologies Used

  • HTML5 – Page structure
  • CSS3 & Bootstrap 5 – Styling and responsiveness
